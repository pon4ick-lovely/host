# UrTube - Simple Video Streaming Platform

UrTube is a simple video streaming platform written in Python. It allows users to register, log in, add videos, search for videos, and watch videos.

## Installation

To install UrTube, follow these steps:

1. Clone the repository:
```bash
git clone https://github.com/your-username/ur-tube.git

2.
Navigate to the project directory:
cd ur-tube

3.
Install the required dependencies:
pip install -r requirements.txt

Usage
To use UrTube, run the main.py file:
python main.py

Here are some basic commands to interact with UrTube:

Register a new user: register <nickname> <password> <age>
Log in as an existing user: log_in <nickname> <password>
Log out: log_out
Add a new video: add <title> <duration> [adult_mode] (optional)
Get a list of videos containing a keyword: get_videos <keyword>
Watch a video: watch_video <title>
